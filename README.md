This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

To run storybook:

```bash
npm run storybook
```

To run test:

```bash
npm run test
```

To generate components:

```bash
npm run generate
```

To run test:

```bash
npm run test
```
